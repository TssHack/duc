GPT API Documentation

Welcome to the GPT API documentation. This API allows developers to generate text responses using custom queries.

API Endpoints

Method Endpoint Description
GET /ehsan/g Fetch text response using query parameters
POST /ehsan/g Fetch text response using JSON body

Parameters

Parameter Type Required Description
q String Yes The query text for the GPT API
userId String Yes Unique user identifier
network Boolean No Enable or disable network requests
license String Yes Your purchased license key (Contact developer for details)
stream Boolean No Enable or disable response streaming

Example Requests

Using cURL (GET)

curl -X GET "https://api-ehsan-gpt4.onrender.com/ehsan/g?q=hi&userId=12345&network=true&withoutContext=false&stream=true&license=YOUR_LICENSE_KEY"

Using cURL (POST)

curl -X POST "https://api-ehsan-gpt4.onrender.com/ehsan/g" -H "Content-Type: application/json" -d '{"q":"Hello","userId":"12345", "license":"YOUR_LICENSE_KEY", "stream": true}'

Using Python (requests)

import requests response = requests.post("https://api-ehsan-gpt4.onrender.com/ehsan/g", json={"q": "Hello", "userId": "12345", "license":"YOUR_LICENSE_KEY", "stream": True}) print(response.json())
